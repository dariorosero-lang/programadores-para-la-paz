La ruta POST /reporte permite recibir información enviada por un usuario mediante una solicitud HTTP POST. En este caso, recibe un mensaje desde el cuerpo de la solicitud y devuelve una respuesta en formato JSON indicando que el reporte fue recibido correctamente.

Esta ruta podría servir en una aplicación comunitaria para que las personas envíen reportes, alertas o información importante, como problemas de seguridad, daños en servicios públicos o situaciones de emergencia. De esta manera la información puede organizarse y almacenarse de forma más eficiente.
